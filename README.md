## API Endpoints
The following API endpoints are available:

GET /api/contacts: Retrieve all contacts.
POST /api/import: Bulk import contacts via XML.
POST /api/contacts/:id: Update a contact.
DELETE /api/contacts/:id: Delete a contact.
